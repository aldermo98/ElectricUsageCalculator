# euc

## Required development tools
- JDK 11 
- Maven 3.6 or above
- Must set Environment Variable "JAVA_HOME" to jdk11 directory. ensure version is set by entering java -version in command prompt

## How to build?
mvn clean package

## How to run?
java -jar target/euc-1.0-SNAPSHOT-jar-with-dependencies.jar

*If this application was downloaded on Windows, simply execute the run.bat file
